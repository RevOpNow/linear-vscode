# Support

## Getting Help

If you need help with the Linear Connector extension, here are your options:

### Bug Reports & Feature Requests

File an issue on the GitHub repository:
https://github.com/RevOpNow/linear-vscode/issues

Please include:
- VS Code version
- Extension version
- Steps to reproduce the issue
- Any relevant error messages from the Output panel (select "Linear Connector" from the dropdown)

### Questions & Discussions

For general questions or discussion, use GitHub Discussions:
https://github.com/RevOpNow/linear-vscode/discussions

### Linear API Issues

If your issue is related to the Linear API itself (e.g. rate limits, data not syncing), check the Linear API documentation:
https://developers.linear.app

## Requirements

- VS Code 1.85 or later
- A Linear account with a Personal API key
